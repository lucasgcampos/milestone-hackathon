# Milestone

Projeto web 3 para criação de vaquinhas

## Como rodar o Projeto

### Executar os comandos:

#### Anvil
1. anvil -b 5

#### Backend
1. cd milestone-api
2. docker-compose up mongodb
3. yarn
4. yarn dev

#### Frontend
1. cd milestone-web
2. yarn
3. yarn dev
